# gcli
Helper for conventional commits

## Instalation

```curl -s https://raw.githubusercontent.com/daflecardoso/gcli/main/installer.sh | bash -s```

## Setup

create a ```gcli.json``` file root project

```json
{
  "name": "name",
  "color": "#1476FF",
  "showTutorial": false,
  "scopes": [
    "home",
    "products",
    "foo",
    "bar"
  ]
}
```

## Usage

```gcli```

## Update gcli

```gcli --update```
